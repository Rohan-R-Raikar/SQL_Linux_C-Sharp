Linux

Linux is an open-source, Unix-like operating system. Common distributions include Ubuntu, CentOS, Debian, and Red Hat.

File System

The Linux file system follows a hierarchical structure starting from /.

  • /etc/: Contains configuration files.
  • /var/: Stores variable data like logs and caches.
  • /bin/ and /sbin/: Store essential system binaries.
